---

To clone the repository:

```cmd
git clone https://github.com/victorradael/github_explorer.git
```

To install the dependencies use the command

```cmd
npm install
```

or, if you use **Yarn** use the command

```cmd
yarn
```

To run the project:

```cmd
yarn start
```
